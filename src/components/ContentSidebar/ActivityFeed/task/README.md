## Task

Task primitive

### Usage
```js
    import { Task } from 'box-react-ui';

    ...

    <Task />
```

#### Options
- **`createdAt: string` [optional]**: Task created date
- **`text: string`**: Task text
- **`user: string`**: Task user
- **`assignees: string`**: Task assignees
- **`dueDate: number`**: Task due date
